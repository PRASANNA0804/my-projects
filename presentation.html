URL Web Crawling

Ingest the web. One URL at a time.

🔗

Paste URL

Any public http/https page — docs, wikis, articles, company sites

→
🕷️

BFS Crawl

Depth 1–3 control. Follows same-domain links only. Max-pages cap prevents runaway crawls

→
🧹

Clean Extract

Trafilatura strips ads, nav, footers. BeautifulSoup fallback for JS-heavy sites

→
📡

Live SSE Progress

Each page streams a progress event — watch the crawl happen in real time

→
🗃️

Into ChromaDB

Chunked, embedded via Azure text-embedding-ada-002, stored with URL metadata

⚙️

Depth Control

Depth 1 = single page. Depth 2 = page + all its links. Depth 3 = two levels deep. Max 50 pages per crawl.

🛡️

Smart Bot Protection

Known blocked sites (Twitter, LinkedIn, paywalls) are detected instantly and skipped with a clear error message — no wasted time.

📖

Wikipedia Native API

Wikipedia URLs use the REST API directly — bypasses bot-blocking, gets clean article text without scraping.
